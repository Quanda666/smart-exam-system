# 接口设计记录

本文件用于持续记录在线考试系统接口设计。当前阶段只记录阶段 1 最小接口。

## 通用响应结构

```json
{
  "success": true,
  "code": "OK",
  "message": "操作成功",
  "data": {}
}
```

## 阶段 1 接口

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | /api/health | 后端健康检查 |
| GET | /api/ai/status | AI 配置状态检查 |
