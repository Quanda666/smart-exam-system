# 05 教师端开发说明

## 题库

- 支持题目草稿、提交审核、发布、撤回、归档。
- 支持题目版本，已被试卷引用的题目不得直接覆盖历史内容。
- 支持单选、多选、判断、填空、主观题；填空题配置精确匹配、忽略空格、多个可接受答案。
- AI 生成题必须保留来源、模型、提示词版本和人工确认人。

## 试卷

- 手动组卷：按题型、知识点、难度筛选题目。
- 规则组卷：指定题型、知识点、难度、题量、分值，可生成多套等价试卷。
- 发布考试时生成试卷快照，后续题库修改不影响已发布考试。
- 已发布考试引用的试卷进入锁定状态，只允许复制后再改。

## 考试发布

- 发布前校验：试卷已发布、总分大于 0、时间合法、时长合法、参考范围非空。
- 发布后生成考生快照，后续班级人员变化不影响本次考试，除非教师手动同步。
- 支持撤回未开始考试、提前结束进行中考试、导出考生名单和成绩单。

## 监考

- 进行中考试看板显示：应到、已进入、作答中、已交卷、异常人数。
- 查看单个学生事件：切屏、失焦、复制、粘贴、退出全屏、断线、超时。
- 监考事件只展示风险，不自动扣分或判违规。

## 阅卷

- 待阅卷列表按考试、学生、剩余题数展示。
- 每题显示题干、学生答案、参考答案、最大分值、AI 建议、历史评语。
- 分数不得超过题目分值。
- 支持退回重评、复评、仲裁的扩展字段和流程预留。

## 学情分析

- 按考试、课程班、知识点统计平均分、及格率、得分率、错题排行。
- 单个学生画像展示考试趋势、知识点掌握、错题分布、异常事件摘要。
