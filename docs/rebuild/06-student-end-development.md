# 06 学生端开发说明

## 考试中心

- 分类展示：未开始、可进入、作答中、已交卷、待批阅、已出分。
- 未开始考试进入候考页，展示规则、时间、次数、设备检查。
- 可进入考试必须先确认考试须知，再创建或恢复作答会话。

## 作答页

- 使用专用全屏/沉浸作答布局，不显示普通业务菜单。
- 服务端返回剩余时间，刷新不得重置倒计时。
- 自动保存服务端草稿，本机保留离线草稿作为兜底。
- 支持未作答题目导航、保存状态、断线提示、交卷确认。
- 到时自动交卷，交卷失败必须保留本机草稿并提示重试。

## 监考提示

- 学生端不恐吓式打断作答，只在必要时提示“已记录异常操作”。
- 切屏、失焦、复制、粘贴、退出全屏、网络断连都上报为风险事件。
- 风险事件不在学生端显示细节裁决，处理结果由教师或管理员确认。

## 成绩与反馈

- 未发布成绩只显示“已交卷/待批阅/待发布”，不显示分数和答案。
- 已发布后展示总分、题目得分、正确答案、解析、教师评语。
- 支持成绩申诉，申诉需要选择题目、填写原因、上传附件。
- 错题本按考试、知识点、题型筛选，支持错题重练。

## 学习推荐

- 根据错题和知识点得分率生成复习建议。
- AI 讲解只基于已发布成绩和错题内容，不暴露未发布答案。
