# 192. 管理员通知审计结果复制链接

## 背景

教师端已经可以复制通知审计链接，管理员端也能通过深链定位通知审计记录。管理员确认审计结果后，仍只能复制通知 ID，不方便把确认后的记录链接回传给教师或写入留档。

## 本批改动

- 管理员端 `Notification Audit` 结果表的 ID 列增加复制审计链接按钮。
- 复制链接复用共享 `copyNotificationAuditLinkToClipboard` 工具。
- ID 列宽度从 `120` 调整为 `150`，容纳 ID、复制 ID、复制链接三个元素。
- 质量门禁增加管理员端复制链接按钮和共享工具调用断言。

## 三端协同影响

- 管理员端：审计确认后可以直接复制可复现的通知审计链接。
- 教师端：收到管理员回传链接后，可以看到同一条通知审计上下文。
- 学生端：不改变通知读取、考试作答或监考事件上报流程。

## 验收点

- `Notification Audit` 结果行存在 `Copy notification audit link` 按钮。
- 点击后复制 `/monitor/logs?notificationId=<id>` 格式链接。
- 既有复制 ID 行为不变。
- 打开复制链接后仍可预填并定位通知 ID。
