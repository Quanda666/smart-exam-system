# 前端应用

本目录是智慧在线考试与学习反馈系统前端，使用 Vue 3、Vite、TypeScript、Element Plus 和 ECharts 实现管理员端、教师端、学生端三端业务界面。

## 环境要求

- Node.js 18+
- npm 9+

## 启动

```bash
npm install
npm run dev
```

默认访问地址：`http://127.0.0.1:3000`。

也可以在项目根目录运行：

```cmd
scripts\run-frontend.cmd
```

前端开发服务器代理 `/api` 到 `http://localhost:8080`。

## 构建

```bash
npm run build
```

生产构建产物位于 `frontend/dist/`。推荐通过后端托管或静态服务器访问，不建议直接双击 `index.html`。

一体化本地运行可使用：

```cmd
scripts\run-all.cmd
```

该脚本会将 `dist` 同步到 `backend/src/main/resources/static`，最终通过 `http://localhost:8080` 访问完整系统。

## 当前页面

- 登录、注册、邮箱验证码登录、邮箱绑定、个人资料与密码修改。
- 三端概况：管理员、教师、学生登录后默认进入各自概况页。
- 管理员：基础数据、题库、试卷、考试任务、成绩分析、用户管理、角色管理、系统日志。
- 教师：题库、AI 出题台、试卷、考试任务、阅卷、基础数据、成绩分析、学生画像。
- 学生：考试中心、在线答题、成绩记录、错题本、知识点掌握度、错题 AI 讲解、公告。
- 通知中心：未读数量、已读标记、全部已读。

## AI 出题台

题库管理页集成三种 AI 工作流：

- 直接生成：选择科目、知识点、题型、难度和数量，生成题目草稿。
- 题目文档导入：上传 `txt`、`md`、`docx`、`doc`、`pptx`、`ppt`、`pdf` 等文档，自动识别题干、选项、答案和解析。
- 课程资料生成：上传课程资料，按题型分别设置数量，由 AI 根据资料生成关联知识点的题目。

所有 AI 结果都会先进入预览与微调环节，教师确认后才保存为题库草稿。

## 配置

| 变量 | 说明 |
|---|---|
| `VITE_API_BASE_URL` | 分离部署时配置后端 API 公网地址；同源部署时留空 |

开发环境默认通过 Vite proxy 访问后端。云端一体化部署时，前端静态资源由 Spring Boot 托管。
