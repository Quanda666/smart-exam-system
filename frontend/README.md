# 前端应用

本目录为智慧在线考试与学习反馈系统前端应用，当前使用 Vue 3 + Vite + TypeScript + Element Plus 实现阶段 10，已包含所有核心功能和 AI 辅助。

## 环境要求

- Node.js 18+
- npm 9+

## 启动命令

```bash
npm install
npm run dev
```

默认访问地址：

```text
http://127.0.0.1:3000
```

Windows 环境也可在项目根目录使用：

```cmd
scripts\run-frontend.cmd
```

阶段 1 中，Windows 对 `5173` 端口绑定返回权限拒绝，因此前端开发服务默认改为 `127.0.0.1:3000`。

## 当前页面

当前前端页面已包含：

1. 登录页，支持管理员、教师、学生三类演示账号。
2. 登录后按角色展示不同侧边菜单和默认首页。
3. 管理员、教师、学生三类工作台雏形。
4. 本地 Token 保存和退出登录。
5. 权限隔离验证按钮，用于演示非当前角色接口会被后端拒绝。
6. 基础资料管理页面，支持班级、科目、知识点和公告列表、筛选、新增、编辑、删除。
7. 题库管理页面，支持题目统计、多条件筛选、动态题型表单、选项维护、发布撤回和删除。
8. 试卷管理页面，支持手动选题组卷、按规则自动组卷、试卷预览、发布撤回和删除。
9. 教师端考试任务页面，支持创建考试、关联试卷和班级、设置时间。
10. 学生端考试中心页面，展示待考、进行中和已完成的考试列表。
11. 学生端在线答题页面，支持倒计时、题目展示、答案选择/填写和提交。
12. 教师端阅卷页面，支持对主观题进行打分和评论。
13. 学生端成绩查询、错题本和知识点掌握度分析页面。
14. 管理员端日志查询和异常事件查看页面。

## 题库管理能力

| 能力 | 说明 |
|---|---|
| 题型 | 单选题、多选题、判断题、填空题、主观题 |
| 筛选 | 关键词、科目、知识点、题型、难度、状态 |
| 表单 | 客观题动态维护选项和正确答案，非客观题维护参考答案 |
| 状态 | 草稿、已发布，支持发布与撤回 |
| 权限 | 管理员和教师可维护，学生不可访问题库管理页 |

## 试卷管理能力

| 能力 | 说明 |
|---|---|
| 手动组卷 | 从已发布题库中按科目筛选并手动选题、设置分值 |
| 规则组卷 | 按科目、知识点、题型、难度、数量、分值配置规则，自动抽题组卷 |
| 预览 | 查看试卷题目、总分、题量 |
| 状态 | 草稿、已发布，支持发布与撤回 |
| 权限 | 管理员和教师可维护 |

## 考试与答题能力

| 能力 | 说明 |
|---|---|
| 创建考试 | 教师可选择已发布的试卷，指定考试班级、起止时间和时长 |
| 学生考试列表 | 学生可查看个人待考、进行中、已完成的考试 |
| 在线答题 | 在规定时间内进入考试，系统提供倒计时和题目导航 |
| 自动保存 | （待实现）切换题目或定期自动保存学生答案 |
| 提交试卷 | 提交后客观题自动评分，主观题进入待批阅状态 |

## 角色页面差异

| 角色 | 可见基础资料菜单 | 写操作范围 |
|---|---|---|
| 管理员 | 班级、科目、知识点、公告 | 可维护全部基础资料 |
| 教师 | 科目、知识点、公告 | 可维护科目、知识点、公告，可查看班级 |
| 学生 | 公告 | 只读公告和后续学习相关数据 |
