# 前端应用

本目录为智慧在线考试与学习反馈系统前端应用，阶段 1 先提供 Vue 3 + Vite + TypeScript + Element Plus 最小骨架。

## 环境要求

- Node.js 18+
- npm 9+

## 启动命令

```bash
npm install
npm run dev
```

默认访问地址：

```text
http://127.0.0.1:3000
```

阶段 1 中，Windows 对 `5173` 端口绑定返回权限拒绝，因此前端开发服务默认改为 `127.0.0.1:3000`。

## 阶段 1 页面

阶段 1 只提供项目首页、后端健康检查卡片和 AI 状态卡片，用于验证前后端接口连通。
